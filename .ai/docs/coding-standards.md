# Padrões de Código e Comportamento

1. **Idioma:** Responda sempre em Português no chat.
2. **Commits:** Gere mensagens de commit SOMENTE em Inglês, seguindo o padrão Conventional Commits (ex: `feat: add data validation`).
3. **Documentação:** A cada alteração de código, verifique se a documentação em `.ai/docs/` precisa ser atualizada. Mantenha docstrings atualizadas.
4. **Infraestrutura:** Foco em Mínima Infraestrutura Viável. Não sugira serviços complexos de nuvem se um container Docker simples resolver. Evite over-engineering.
5. **Debugging_summary.md:** Mantenha o arquivo Debugging_summary.md atualizado conforme o indicado no cabeçalho do arquivo, sempre atualize o Debugging_summary.md ao  detectar novos arquivos, arquivos modificados ou debuggs realizados.
